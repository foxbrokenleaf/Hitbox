## 制板
### 热转印法
#### 工具材料
+ 热转印机/熨斗
+ 热转印纸
+ 激光打印机
+ 覆铜板
+ 砂纸/钢丝球
+ 蚀刻剂
#### 热转印
使用激光打印机将镜像的PCB打印到转印纸（反光面）上，然后使用热转印机或者熨斗按压转印，然后缓慢揭开热转印纸
![20260303_235450](https://github.com/user-attachments/assets/16725a4b-3ee1-481f-97e9-14ebd41fef98)
#### 蚀刻
配置蚀刻液，将转印好的PCB放入蚀刻液中，确保溶液没过PCB板
![20260304_000608](https://github.com/user-attachments/assets/b8e93b61-8030-4109-9672-1f3b7de6e9f5)
